Questions and answers

What distinguishes normal Html from semantic html

normal Html elements describe the structure of a webpage without conveying a specific meaning while Semantic Html elements provide meaning and context to the content they surround.

Explain the disparity between closing tags and self closing tags

Closing tags are used to mark the end of an html elements while self closing tags are used for elements that dont have any content within them.

What distinguishes inline elements and block elements

Inline elements are elements that dont start on a new line and only take up as much width as necessarry while block elements start on a new line and take up the fll width available.
